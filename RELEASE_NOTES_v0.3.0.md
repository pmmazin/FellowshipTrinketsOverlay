# Fellowship Trinkets Overlay v0.3.0

## Nouveautes

- Ajout du suivi des interrupts/kicks depuis les logs de combat.
- Affichage du joueur qui kick, du sort interrompu et du cooldown du kick.
- Detection des kicks utilises meme quand ils ne coupent aucun sort.
- Ajout des icones officielles des sorts de kick connus.
- Ajout d'un halo de curseur reglable pour mieux voir la souris en jeu.
- Ajout d'un bouton de mise a jour directement dans les reglages.
- Ajout du choix du dossier d'installation.
- Les futures mises a jour peuvent remplacer l'installation existante et relancer l'application automatiquement.

## Ameliorations

- Le build portable inclut maintenant les icones des kicks.
- Le script de build genere automatiquement l'archive ZIP a publier dans GitHub Releases.
- La mise a jour telecharge les fichiers en streaming pour eviter les timeouts courts pendant le telechargement.

## Important

Les utilisateurs qui ont une ancienne version sans bouton de mise a jour doivent installer cette version une derniere fois manuellement.
Apres cette version, les mises a jour suivantes pourront etre installees depuis l'application.
